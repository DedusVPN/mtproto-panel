Развёртывание Telemt

Доступ по http://IP:порт без шифрования: ограничьте порт файрволом/VPN, используйте сильный пароль панели. После входа учётные данные SSH идут по WebSocket на тот же хост — не открывайте панель в публичный интернет без защиты сети.

Опции деплоя

Безопасность и шейпер

Аналог настроек из Reshala: UFW, Fail2Ban, sysctl hardening, шейпер tc (скачивание: egress; загрузка: IFB). Выполняется в конце деплоя.

Скачивание (клиент ← сервер): исходящий HTB по sport. Загрузка (клиент → сервер): ingress → ifb0, HTB по dport.

Провайдеры

VDSina

Тариф-конструктор: если заказ отклоняется, задайте CPU / RAM (ГБ) / диск по лимитам тарифа в API.

Токен: VDSINA_API_TOKEN (без слова Bearer). По умолчанию API: https://userapi.vdsina.com/v1; для кабинета на vdsina.ru задайте VDSINA_API_BASE=https://userapi.vdsina.ru/v1. Также поддерживается VDSINA_BASE_URL. Файл .env в корне проекта; переменные среды перекрывают .env. «В панель» создаёт новую запись в servers.json с root и паролем из API. При 401 — /api/cloud/vdsina/status (token_diagnostics). Документация API.
